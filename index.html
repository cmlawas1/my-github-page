Previous Projects

Ang Mafall, Talo! is a shooter game project coded using Java. The game's objective is to shoot the most number of boys with your bullet hearts and collecting item boosters while avoiding to lose health points. I worked on all front-end designs and codes, and helped fix bugs in the back-end.

This solo project allows a user to input an initial matrix tableau and solve it through simplex methods. It also has a polynomial quadratic spline interpolator, which can interpolate a value from a given set of x and y points. It is coded using R and the Rshiny package.

Adopt N' Shop is a proposed mobile app for putting pets up for adoption, adopting pets, and shopping items for them in the same platform. This app has not been programmed, but I designed the app mockup including the logo.

Splitwise is a program that allows you to split expenses with your friends with ease. Together with my teammates, we integrated SQL (MariaDB) and Python to be able to view, add, update, and delete a user's expenses made with a friend or a group.
